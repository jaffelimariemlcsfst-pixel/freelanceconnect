# Workshop Rules & Guidelines

## Technology Stack
- **Frontend**: ALWAYS use Next.js with Tailwind CSS.
- **Backend**: ALWAYS use FastAPI (Python).
- **Database**: Use firestore as a database

## Code Style
- Follow PEP 8 for Python.
- Use Prettier for JavaScript/TypeScript/CSS.

## Workshop Constraints
- Keep implementations simple and focused on the learning objective.
- Always include comments explaining complex logic for educational purposes.
