# 2️⃣ Install Python

Python is required for the FastAPI backend.

### Steps

1. Go to: https://www.python.org

2. Download the latest stable version

3. ⚠️ ⚠️ ⚠️ IMPORTANT: Check “Add Python to PATH”

4. Install

5. Verify:

```bash
python --version
pip --version
```
✅ If version numbers appear, you're good.
